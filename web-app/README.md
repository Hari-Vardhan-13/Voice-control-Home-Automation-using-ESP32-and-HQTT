# Web App (local reference version)

This `index.html` is a single-file, no-build voice-control client — useful
for quick local testing (just open it in a browser) and as a reference for
exactly which MQTT topics/payloads a voice front end should use.

**For the actual demo/submission, use the deployed version instead:**
https://home-automation-voice-control.netlify.app/

See `../docs/deployment.md` for how the two relate, and the project root
`README.md` for the note on what to double check on the deployed app.
