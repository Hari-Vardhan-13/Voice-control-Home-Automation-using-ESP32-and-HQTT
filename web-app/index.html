🏠 Voice Control Home Automation

Speak a command, or tap a device. Connects directly to your HiveMQ broker over WebSocket.
Not connected
Try: "turn on the bedroom light" / "turn off the fan" / "open the door"
Activity log
Setup: In HiveMQ Cloud Console, enable the WebSocket listener (default port 8884) — it's on by default for new clusters. Paste that URL above along with your existing MQTT username/password. This page never talks to the ESP32 directly; it publishes to the same broker topics (home/<device>/set) the firmware already subscribes to.

Privacy: speech recognition runs via your browser's built-in Web Speech API (Chrome/Edge). Nothing is sent anywhere except the short MQTT command.
